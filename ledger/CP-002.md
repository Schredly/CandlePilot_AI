## Decisions & notes
- **Did not unify TrendBadge / ConfidenceBar / DeltaText.** These render visually different pills/bars per page by Figma's intent (Scanner pill has a border, Dashboard pill doesn't; IntelligencePanel bar is violet→fuchsia, TopSignals bar is blue→purple). Unifying would flatten the design. A single component with three boolean props is worse than three inline instances.
- **Did not extract scanner / backtesting / alerts card containers.** They use different color families (`bg-[#0d0d12]`, `bg-slate-900/50`, shadcn `Card`). Each page has its own intentional visual identity; extracting a "universal card" would erase it.
- **Scope limit.** CP-002 was a refactor sprint riding on already-shipped code. Kept the diff surface narrow so no visual regression is possible.
